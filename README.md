My Portfolio
A fully editable, full-stack personal portfolio website built with Node.js, Express, and TailwindCSS.

Dark ThemeNode.js

✨ Features
🔒 Password-Protected Edit Mode — Securely edit your portfolio from the browser.
✍️ Dynamic Typing Animation — Customizable title/tagline.
📬 Contact Form Inbox — Receive and view messages securely.
📊 Visitor Counter — Tracks page visits.
📁 Export/Import JSON — Backup and restore your data easily.
🖨️ Print / PDF Mode — Clean resume printing.
📱 Fully Responsive — Looks great on mobile, tablet, and desktop.
🚀 How to Run Locally
Clone the repository:
git clone https://github.com/mnaveenn850-create/portfolio.git
Navigate to the folder:
bash
cd portfolio
Install dependencies:
bash

npm install
Start the server:
bash

node server.js
Open your browser and go to: http://localhost:3000
🔑 Default Password
The default edit password is admin123.
(You can change this in the server.js file under const PASSWORD)

🛠️ Tech Stack
Frontend: HTML, TailwindCSS, JavaScript
Backend: Node.js, Express
Storage: File-based JSON (Local)
